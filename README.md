## 使用方法

### 构建并运行项目

```bash
// 克隆此项目
git clone https://github.com/zzXiongFan/my-blog.git
// 进入项目并安装相关依赖
cd my-blog
yarn install  // vuepress推荐使用yarn,，若您习惯使用npm，可以使用命令：npm install
// 运行项目
yarn run dev // npm run dev
```
